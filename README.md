## 🚀 الرفع على Netlify (بدون Supabase — وضع محلي كامل)

1. سجّل دخولك على [netlify.com](https://netlify.com)
2. اضغط **Add new site → Deploy manually**
3. اسحب مجلد المشروع كاملاً وأفلته
4. انتظر دقيقة واحدة → سيعطيك رابط جاهز! ✅

**أو عبر Git:**
```
git init && git add . && git commit -m "v3"
# ثم اربط الـ repo بـ Netlify
```

---

## ⚙ المتغيرات البيئية (اختياري)
في Netlify → Site Settings → Environment Variables:
```
VITE_SUPABASE_URL    = https://xxxx.supabase.co
VITE_SUPABASE_ANON_KEY = xxxx
```
**بدون Supabase:** البيانات تُخزن في localStorage للمتصفح — تعمل بشكل كامل.
